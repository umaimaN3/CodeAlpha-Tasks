🚀 How to Run the Project

Follow these steps to run the project:

---

🔧 Backend Setup

1. Install dependencies

npm install

---

2. Start MongoDB (if using local database)

mongod

 make sure ".env" file already has correct connection string.»

---

3. Start backend server

npm start

You should see:

MongoDB Connected
Server running on port 5000

---

🌐 Frontend Setup

Open frontend:

- Open "index.html" in browser
  OR
- Use VS Code Live Server

---

⚠️ Important

- Backend must be running before opening frontend
- MongoDB must be running (if using local DB)

---

✅ Done

Now you can:

- Create projects
- Add tasks
- Update task status
- Add comments

---
